使用方法

这是一个说明书

如果没有的话可以百度下载几个，我在文件夹里面也准备了几个。
  

2.        这里可以设置字体的基础大小。

3.        输入一些随机参数，这个决定了字体的畸变.

4.        输入页面参数，字间距由列间距决定.

5.        输入文本.

6.        开始测试吧！
